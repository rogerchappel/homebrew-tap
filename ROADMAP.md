# Roadmap

## Now

- Maintain a safe HEAD-only catalog for active Roger CLI projects.
- Keep README install snippets synchronized with formulas.
- Validate generated formulas in CI.

## Next

- Add release-backed formulas as upstream projects publish immutable archives.
- Add livecheck blocks once release cadence is established.
- Expand the catalog only after local install tests are repeatable.

## Later

- Consider bottles or casks if the ecosystem grows release infrastructure for them.
- Generate README catalog tables directly from `catalog/tools.json`.
