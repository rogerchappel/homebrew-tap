# Catalog Policy

A tool can enter `catalog/tools.json` when it has:

1. A public `rogerchappel/*` GitHub repository.
2. A CLI entrypoint that can be linked by Homebrew.
3. A repeatable build path on macOS.
4. A clear caveat if it remains HEAD-only.

Do not add private repositories, local-only scripts, or projects that require secrets during installation.
